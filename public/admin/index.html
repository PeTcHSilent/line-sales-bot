🔗 Sync ข้อมูลจาก HR System

⏱️ Sync ล่าสุด: กำลังตรวจสอบ...
🧑‍💼 พนักงานแผนก Sales + Admin จาก HR จะถูก sync มาที่หน้าข้อมูลพนักงานอัตโนมัติ
📅 วันหยุดนักขัตฤกษ์จาก HR Calendar จะถูก sync มาใช้กับการแจ้งเตือน
⏰ ระบบ sync ให้อัตโนมัติทุก 6 ชั่วโมง

ℹ️ เกี่ยวกับการแจ้งเตือน

🔔 Lead ใหม่ — แจ้งเตือนทันทีเมื่อลูกค้าให้ข้อมูลครบ

🔄 ลูกค้าต่ออายุ — แจ้งเตือนเมื่อลูกค้าตอบรับการต่ออายุ

🙋 รับงาน — แจ้งเตือนพนักงานทุกคนเมื่อมีคนกดรับงาน

💡 จัดการข้อมูลพนักงานและ LINE ID ได้ที่เมนู 🧑‍💼 ข้อมูลพนักงาน

📱 วิธีดู LINE User ID ของพนักงาน

1. พนักงานเปิด LINE → ค้นหา Official Account บริษัท
2. พิมพ์: !myid
3. Bot ตอบกลับ LINE User ID — นำมากรอกในช่อง LINE ID ของพนักงานคนนั้น
